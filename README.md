# ☕ HAG Coffee Website

## 📝 Giới thiệu
Website bán cà phê đơn giản được xây dựng bằng HTML, CSS và JavaScript. Dự án tập trung vào việc tạo ra giao diện thân thiện và quy trình đặt hàng cơ bản cho người dùng.

🚀 **Link Demo:** [https://hieu09093.github.io/HAG_coffe/](https://hieu09093.github.io/HAG_coffe/)

## ✨ Chức năng
* **Hiển thị danh sách sản phẩm:** Trình bày menu đồ uống đa dạng.
* **Đăng ký / Đăng nhập:** Hệ thống quản lý người dùng tích hợp Firebase.
* **Giỏ hàng:** Tính năng thêm sản phẩm vào giỏ và tự động tính tổng tiền đơn hàng.
* **Responsive:** Giao diện tối ưu hiển thị trên nhiều kích thước màn hình khác nhau.

## 🛠 Công nghệ sử dụng
* **HTML, CSS, JavaScript:** Ngôn ngữ nền tảng cho giao diện và tương tác.
* **Firebase Authentication:** Xử lý xác thực người dùng an toàn.
* **LocalStorage:** Lưu trữ dữ liệu giỏ hàng tạm thời trên trình duyệt.
