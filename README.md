# HAG Coffee Website

## Giới thiệu
Website bán cà phê đơn giản được xây dựng bằng HTML, CSS và JavaScript.

## Chức năng
- Hiển thị danh sách sản phẩm
- Đăng ký / đăng nhập người dùng (Firebase)
- Giỏ hàng (thêm sản phẩm, tính tổng tiền)
- Responsive giao diện

## Công nghệ sử dụng
- HTML, CSS, JavaScript
- Firebase Authentication
- LocalStorage
